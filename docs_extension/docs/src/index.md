# ExtDocs

Extension documentation for reconstruction, visualization, compaction-aware interpretation, archive export, batch analysis, and transport-related helpers built around CarboKitten.

## Main workflows

- Reconstruct sections and volumes from model state
- Visualize facies, profiles, Wheeler diagrams, and summary figures
- Build interpreted environment grids from facies volumes
- Export derived products and batch summaries
