# Overview

ExtDocs groups the extension functionality into a small set of domains:

- **Reconstruction**: rebuild surfaces, sections, and data objects from model state
- **Visualization**: create figures from slices, volumes, columns, and archives
- **Production and compaction**: process-level helpers for production and preserved-layer workflows
- **Environment interpretation**: derive interpreted categorical environments from facies data
- **Archive and output**: write derived products and output-oriented helpers
- **Batch analysis**: run summaries and flatten diagnostics for tabular export
- **Transport and wave physics**: advanced transport and hydrodynamic helpers

Use the workflow pages first, then the reference pages for detailed API documentation.
